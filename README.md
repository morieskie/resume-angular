<h2>ğŸ› ï¸ Installation Steps:</h2>

<p>1. Clone repo</p>

```
git clone https://github.com/morieskie/resume-angular.git
```

<p>2. Go into project root folder</p>

```
cd resume-angular
```

<p>3. Install dependencies</p>

```
npm install
```

<p>4. Run unit tests</p>

```
npm test
```

<p>5. Building the project bundles</p>

```
npm run build
```

```
npm start
```
